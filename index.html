Commandes GitLab Essentielles

Cloner un dépôt GitLab :

Utilisez cette commande pour cloner un dépôt GitLab localement.

git clone https://gitlab.com/username/projet.git
Ajouter des fichiers et préparer un commit :

Ajoutez tous les fichiers modifiés dans l'index pour préparer un commit.

git add .
Créer un commit :

Créez un commit avec un message explicatif.

git commit -m "Message de commit"
Pousser les modifications vers le dépôt distant :

Envoyez vos modifications locales sur le dépôt GitLab distant.

git push origin nom-branche
Récupérer les modifications du dépôt distant :

Synchronisez votre dépôt local avec les dernières modifications depuis GitLab.

git pull origin nom-branche
Afficher le statut des fichiers :

Vérifiez les fichiers modifiés, non suivis ou en attente de commit.

git status
